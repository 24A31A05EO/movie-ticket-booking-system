# 🎬 Movie Ticket Booking System for Multiplex Theatre

## 📌 Project Title
Website for Movie Ticket Booking System (Frontend Project)

## 📝 Project Description
This project is a frontend-based web application designed for a multiplex theatre ticket booking system.  
It allows users to register, log in, browse movies, and select seats through a simple and user-friendly interface.

⚠️ Note: This project is built using only frontend technologies and does not include backend or database integration.

## ❗ Problem Statement
Traditional offline ticket booking can be time-consuming due to long queues.  
This project provides an online interface prototype to simplify movie browsing and seat selection, improving user experience.

## ✨ Features of the Project
- User Registration and Login Interface
- Display of available movies
- Seat selection layout
- Responsive and clean UI
- Smooth navigation between pages
- Basic interactivity using JavaScript

## 🛠️ Technologies Used
- HTML – Structure of web pages  
- CSS – Styling and layout  
- JavaScript – Interactivity  
- Tools – Visual Studio Code / Notepad++  
- Browser – Google Chrome / Microsoft Edge

## 📂 Project Structure
Movie-Ticket-Booking-System
│
├── 📄 index.html        # Home page
├── 📄 login.html        # Login page
├── 📄 register.html     # Registration page
├── 📄 movies.html       # Movies listing page
├── 📄 seats.html        # Seat selection page
├── 🎨 style.css         # Styling
└── ⚙️ script.js         # JavaScript logic

## ⚙️ Installation / Setup
1. Download or clone the repository  
2. Open the folder in VS Code or any editor  
3. Open `index.html` in browser  
4. Ensure all files are in the same folder  
5. No backend setup required  

## 🚀 Usage of the Project
- Open Home Page (`index.html`)  
- Register a new account (UI only)  
- Login using credentials  
- Browse available movies  
- Select a movie and choose seats  
- Confirm booking (simulation)
